Support independent Russian media

About the project

Inspired by, and based on @detoxbrainwash video.

In parallel with the war in Ukraine, another war happens inside Russia between Putin and Russian people who are not willing to support Putin's regime.
In the situation when most independent media are destroyed and social networks are blocked, YouTube becomes one of the few remaining platforms where people can share their opinion and get access to the independent sources of information.

According to @detoxbrainwash, one of the strategies used by Putin involves boosting propaganda materials on YouTube. As a result, pro-government videos dominate in YouTube recommendations.

We are providing a Chrome Extension that can help to boost selected videos and YouTube channels, if used collectively.
The extension will open playlist in multiple tabs and loop through videos with optimal interval. Supported browsers are Chrome, Edge and Opera.

How to install the extension

If you are comfortable using Chrome account and installing extension from the Chrome Store, use this link and follow standard installation procedure.

Otherwise you can install the extension locally. Select browser to see the instructions:

  1. Download extension folder from GitHub - use this link.
  2. Unzip the archive in any location on your machine.
  3. Open Extentions manager in Chrome browser - copy this link to your browser chrome://extensions/
  4. Enable developer mode in the top right corner - see the screenshot.
  5. Click "Load unpacked" button in the top left corner to upload extension, select the extension folder.
  6. Extension is ready to use. To bookmark the extension, click "Extensions" button in the top right corner in the browser, the "Pin" icon next to extension name.
  7. It is recommended to use YouTube AdBlock extension to skip ads.

Then click the extension button, and select the number of tab that works for you.
